# Praktikum Git - Website Rental Game

## A. Deskripsi Proyek

Website ini merupakan **landing page rental game** bernama **SteamRent** yang menampilkan daftar game populer beserta harga dan informasi singkat.
Website dibuat menggunakan **HTML, CSS (Bootstrap), dan JavaScript** dengan fitur tambahan seperti **toggle Bright/Dark Mode**.

---

## B. Cara Menjalankan

1. Clone repository:

   ```bash
   git clone https://github.com/istmaa/praktikum-git-25-564956-SV-26891.git
   ```
2. Masuk ke folder project
3. Buka file `index.html` di browser

## D. Dokumentasi Perintah Git

| Perintah      | Fungsi                           |
| ------------- | -------------------------------- |
| git init      | Inisialisasi repository          |
| git clone     | Mengambil repository dari GitHub |
| git add       | Menambahkan file ke staging area |
| git commit -m | Menyimpan perubahan              |
| git push      | Mengirim ke GitHub               |
| git pull      | Mengambil update terbaru         |
| git branch    | Membuat / melihat branch         |
| git checkout  | Pindah branch                    |
| git merge     | Menggabungkan branch             |
| git rebase    | Merapikan commit                 |
